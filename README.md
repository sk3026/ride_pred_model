# 🚖 Ride Demand Prediction System

A machine learning-based system that predicts ride demand using time and location features. The project includes model training, REST API integration, and an interactive dashboard for real-time demand analysis.

---

## 🚀 Features

* Predicts ride demand across locations
* Demand classification: High, Moderate, Low
* REST API for real-time predictions
* Interactive dashboard with visual insights
* Location-based filtering and analysis

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* scikit-learn
* Flask
* Streamlit

---

## 📊 Model Performance

* **R² Score:** 0.95
* **MAE:** 7.07

---

## ⚙️ Project Workflow

1. Data preprocessing and feature engineering
2. Model training and evaluation
3. Flask API development
4. Streamlit dashboard integration
5. Deployment and real-time prediction
